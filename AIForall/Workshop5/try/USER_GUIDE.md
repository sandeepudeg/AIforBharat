## Getting Started

### Accessing the Application

1. Open your web browser
2. Navigate to `http://localhost:5000` (or your server URL)
3. You'll see the homepage with major categories displayed as cards

### Homepage Overview

The homepage features:
- **Search Bar** - Search across all articles
- **Category Cards** - Browse major topics
- **Navigation Menu** - Access different sections
- **Chat Widget** - Ask questions about Pune

## Navigation

### Main Menu

Located at the top of every page:
- **Home** - Return to the homepage
- **About** - Learn about the application

### Category Navigation

1. Click on any category card on the homepage
2. Or use the search bar to find specific topics
3. Use breadcrumbs at the top to track your location

### Breadcrumb Trail

Shows your current location in the site:
- **Home** - Click to return to homepage
- **Category Name** - Current category
- **Article Title** - Current article

Example: `Home > Food & Street Culture > Misal Pav`

## Searching

### Basic Search

1. Click the search bar at the top of the page
2. Type your search query (minimum 2 characters)
3. Press Enter or click the Search button
4. Results appear organized by category

### Search Tips

- **Use specific keywords** - "misal" instead of "food"
- **Search by category** - Results are organized by category
- **Partial matches work** - "pun" will find "Pune"
- **Case insensitive** - "PUNE" and "pune" return same results
- **Multi-word searches** - "street food" searches for both words

### Search Results

Each result shows:
- **Title** - Article name
- **Category** - Which category it belongs to
- **Relevance Score** - How well it matches your query
- **Description** - Brief preview of content

Click any result to read the full article.

## Using Chat

### Opening Chat

1. Look for the **chat icon** (speech bubble) in the bottom-right corner
2. Click to open the chat window
3. Type your question in the input field

### Chat Features

- **Ask Questions** - Ask anything about Pune
- **Get Recommendations** - Receive article suggestions
- **Browse Intent** - Ask to browse categories
- **Search Intent** - Ask to search for topics

### Chat Examples

Try asking:
- "Tell me about Pune's food culture"
- "What are the best places to visit?"
- "Show me information about Puneri culture"
- "What's the weather like in Pune?"
- "Tell me about trekking routes"

### Chat Tips

- **Be specific** - "Tell me about Misal Pav" works better than "food"
- **Ask naturally** - Chat understands conversational language
- **Use follow-ups** - Ask related questions for more info
- **Check articles** - Chat provides relevant article suggestions

### Chat History

- Chat history is saved in your browser
- Clear browser data to reset chat history
- Each session is independent

## Troubleshooting

### Search Not Working

- **Check query length** - Minimum 2 characters required
- **Check spelling** - Verify search term spelling
- **Try different keywords** - Use alternative search terms
- **Clear filters** - Remove category filters if applied

### Article Not Found

- **Check article ID** - Verify the article exists
- **Try searching** - Use search to find the article
- **Browse categories** - Look through category listings
- **Check URL** - Ensure URL is correct

### Chat Not Responding

- **Check message** - Ensure message is not empty
- **Refresh page** - Reload the page
- **Clear cache** - Clear browser cache
- **Try again** - Wait a moment and try again

### Performance Issues

- **Clear cache** - Remove browser cache
- **Disable extensions** - Try disabling browser extensions
- **Try different browser** - Test in another browser
- **Check connection** - Ensure internet connection is stable
