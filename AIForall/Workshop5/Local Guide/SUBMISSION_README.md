## ðŸš€ Project Overview

### What is the Pune Knowledge Base?

A full-stack web application that serves as an intelligent guide to Pune, India. It combines:

- **Interactive Chat Interface** - Ask questions about Pune naturally
- **17 Knowledge Categories** - Geography, Food, Culture, Attractions, History, etc.
- **Full-Text Search** - Find information across all content
- **Responsive Design** - Works on desktop, tablet, and mobile
- **Performance Optimization** - 2x+ speedup through caching
- **Accessibility** - WCAG 2.1 AA compliant
- **Comprehensive Testing** - 224 tests, 100% passing

### Why Pune?

Pune is India's cultural capital with:
- Rich Marathi heritage and traditions
- Diverse street food culture (Vada Pav, Misal Pav, Puran Poli)
- Historical significance (Aga Khan Palace, Shaniwar Wada)
- Unique local slang and expressions
- Vibrant tech and startup ecosystem

### How Does It Understand Local Context?

The application uses `product.md` - a comprehensive local context file that includes:
- Pune's geography and demographics
- Cultural traditions and festivals
- Local food and cuisine
- Historical landmarks
- Local slang and expressions
- Key attractions and places
- Transportation and logistics

This context is used by Kiro to:
- Generate culturally appropriate responses
- Understand Pune-specific queries
- Maintain local tone in chat responses
- Create relevant test cases
- Suggest appropriate design patterns

## ðŸš€ Getting Started

### Installation

```bash
# Clone repository
git clone https://github.com/[username]/pune-knowledge-base.git
cd pune-knowledge-base

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run application
python app.py

# Visit http://localhost:5000
```

### Running Tests

```bash
# Run all tests
python -m pytest tests/ -v

# Run specific test file
python -m pytest tests/test_search_accuracy.py -v

# Run with coverage
python -m pytest tests/ --cov=services --cov=routes
```
