AVAILABLE TOOLS:
================================================================================

1. forecast_demand(sku, sales_data, forecast_days=30)
   → Generates demand forecasts with confidence intervals
   → Stores in DynamoDB

2. optimize_inventory(sku, annual_demand, ordering_cost, holding_cost_per_unit)
   → Calculates EOQ and reorder points
   → Returns optimization recommendations

3. create_purchase_order(sku, supplier_id, quantity, unit_price, delivery_days=7)
   → Creates purchase orders
   → Stores in DynamoDB

4. detect_anomalies(sku, current_inventory, forecasted_inventory, confidence_80, confidence_95)
   → Detects inventory anomalies
   → Stores in DynamoDB
   → Sends SNS alerts if needed

5. generate_report(inventory_data, forecast_data, supplier_data)
   → Generates analytics reports with KPIs
   → Saves to S3

6. get_inventory_status(sku)
   → Gets current inventory level
   → Reads from DynamoDB

HOW IT WORKS:
================================================================================

User Input
    ↓
Strands Agent (supply_chain_orchestrator.py)
    ↓
Analyzes query and decides which tools to use
    ↓
Calls appropriate tools (forecast_demand, optimize_inventory, etc.)
    ↓
Tools execute and store data in AWS
    ↓
Agent returns results to user


DATA FLOW:
================================================================================

Tools → AWS Services → Storage
  ↓
forecast_demand → DynamoDB (forecasts table)
optimize_inventory → Returns calculations
create_purchase_order → DynamoDB (purchase_orders table)
detect_anomalies → DynamoDB (anomalies table) + SNS (alerts)
generate_report → S3 (reports bucket)
get_inventory_status → Reads from DynamoDB


USING IN YOUR OWN AGENT:
================================================================================

from strands import Agent
from strands.models import BedrockModel
from src.agents.agent_tools import (
    forecast_demand,
    optimize_inventory,
    create_purchase_order,
    detect_anomalies,
    generate_report,
    get_inventory_status,
)

model = BedrockModel(model_id="us.anthropic.claude-sonnet-4-20250514-v1:0")

agent = Agent(
    model=model,
    system_prompt="You are a supply chain expert...",
    tools=[
        forecast_demand,
        optimize_inventory,
        create_purchase_order,
        detect_anomalies,
        generate_report,
        get_inventory_status,
    ],
)

response = agent("Forecast demand for PROD-001")
print(response)

TROUBLESHOOTING:
================================================================================

Error: "Unable to locate credentials"
  → Run: aws configure

Error: "ResourceNotFoundException"
  → Run: setup_aws_resources.bat (Windows) or ./setup_aws_resources.sh (macOS/Linux)

Error: "ModuleNotFoundError"
  → Run: pip install -r requirements.txt

Error: "No module named 'strands'"
  → Run: pip install strands-agents
